μLab

Microbiology Practice Lab

Aplikasi interaktif virtual untuk latihan mikrobiologi klinik (S1 Kedokteran)

Tujuan

  • Melatih pengambilan keputusan pemeriksaan mikrobiologi klinik berbasis skenario.
  • Memperkuat konsep dasar: pewarnaan, kultur, identifikasi, dan AST.
  • Memberikan umpan balik instan dan menyimpan progres belajar secara lokal.

Mulai cepat

  1. Buka Learn untuk ringkasan konsep.
  2. Lanjutkan ke Cases untuk latihan berbasis vignette.
  3. Uji diri di Quiz, lalu coba simulasi di Virtual Lab.

Catatan

Versi ini offline-first (tanpa server). Semua data progres disimpan di browser Anda menggunakan localStorage. Cocok untuk GitHub Pages.
